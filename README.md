# 🍎 MeuCardapio - Aplicativo de Gestão Alimentar

[![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev)
[![Dart](https://img.shields.io/badge/Dart-0175C2?style=for-the-badge&logo=dart&logoColor=white)](https://dart.dev)

O **MeuCardapio** é um aplicativo mobile desenvolvido para a disciplina de Programação Móvel. O projeto tem como objetivo auxiliar na organização alimentar e no gerenciamento de cardápios personalizados, permitindo o cadastro de usuários, alimentos e refeições com controle nutricional.

## 🎯 Objetivo do Projeto

Desenvolver um aplicativo mobile para gerenciamento alimentar, permitindo:

- Cadastro de usuários
- Cadastro de alimentos
- Criação de cardápios personalizados
- Consulta de usuários, alimentos e cardápios
- Controle de opções por refeição
- Compartilhamento de informações do aplicativo
- Integração com recursos nativos do dispositivo, como câmera e galeria

---

## 🖥 Tecnologias Utilizadas

- Flutter
- Dart
- SQLite (sqflite)
- Material Design

## 🚀 Como Executar o Projeto

### Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:

- Flutter SDK
- Dart SDK
- Git
- VS Code ou Android Studio

---

### Clone o repositório

```bash
git clone git@github.com:maiaraborsatti/meu-cardapio-app.git
```

---

### Acesse a pasta do projeto

```bash
cd meu-cardapio-app
```

---

### Instale as dependências

```bash
flutter pub get
```

---

### Execute o projeto

```bash
flutter run
```

---

## 📱 Executando em dispositivos

O projeto pode ser executado em:
